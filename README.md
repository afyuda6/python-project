# Python Project

This repository contains a Python GUI application with **currently one implemented feature**:
- **Discount Calculator**.

---

## Feature

- Calculate final price after applying a discount.
- Handle input errors:
  - Empty input
  - Non-numeric input
  - Negative values
  - Discount > 100%
- Show categorized error messages: `Input Error`, `Value Error`.
- Formats final price with thousands separator (e.g., `Rp1,250.00`).

---

## How to Run

1. Clone or download repository.
2. Open terminal or command prompt in the project folder.
3. Run the program:
```bash
python discount_calculator.py
```

---

## Usage

1. Enter the Original Price.
2. Enter the Discount (%).
3. Click Calculate.
4. The Final Price will appears below the button.
5. Popup messages will show if there are invalid input.

---

## Notes

Works on any OS with Python 3.x and Tkinter installed.
